## Spacing System

No CSS variables for spacing — Framer outputs absolute pixel values per breakpoint. Observed rhythm:
- **Section vertical padding**: ~120-160px desktop, ~80px mobile
- **Container max-width**: appears ~1280px with ~80px gutter
- **Card grid**: 3-up at desktop, 1-up at mobile, ~24-32px gap
- **Button padding**: 12px / `border-radius: 0` (squared) — buttons inherit the wordmark's geometry, no rounded pills
