# Jackson & Coil Website2 Prototype

Independent Next.js 15 prototype for Jackson & Coil, built in `/website2/`.

## Run

```bash
npm run dev
npm run lint
npm run build
```

## Stack

- Next.js 15 App Router
- TypeScript
- Tailwind CSS v4
- Framer Motion
- Lenis
- next-intl provider with same-URL client language toggle

See `BUILD_REPORT.md` for scope, design decisions, verification, and known mocked areas.
